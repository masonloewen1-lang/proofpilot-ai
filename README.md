# proofpilot-ai

AI tool that turns rubrics and job postings into execution plans and submission scorecards.

## Simple comparison feature

This repository now includes a lightweight comparison UI for two text inputs (such as job postings or rubrics).

### What it highlights
- Overlapping expectations
- Unique requirements in Input A
- Unique requirements in Input B
- Key differences summary

## Run locally

Open `index.html` directly in your browser, or serve the folder with a static server.
